# Programación Orientada a Objetos en Python

Bienvenido al repositorio principal de la materia de Programación Orientada a Objetos. Este espacio está destinado a organizar y documentar las actividades, ejercicios de laboratorio y proyectos desarrollados a lo largo del curso.

## Objetivo
El propósito de este repositorio es evidenciar el aprendizaje y la aplicación práctica de los pilares del desarrollo de software orientado a objetos, utilizando Python como lenguaje principal.

## Contenido del Repositorio
Cada script o carpeta dentro de este repositorio cuenta con su propio archivo de documentación. De manera general, los ejercicios abarcan los siguientes temas:

* **Fundamentos Básicos:** Creación de clases, instanciación de objetos y uso de métodos constructores.
* **Encapsulamiento:** Manejo de atributos, métodos de acceso y modificación del estado interno.
* **Herencia y Polimorfismo:** Creación de jerarquías, reutilización de código y sobreescritura de métodos.
* **Abstracción:** Diseño de plantillas y métodos obligatorios utilizando el módulo `abc`.
* **Manejo de Excepciones:** Control de errores en tiempo de ejecución mediante bloques `try-except`.
* **Archivos (I/O):** Operaciones de lectura, escritura y manipulación de metadatos con el sistema operativo.

## Ejecución
Para ejecutar cualquiera de los scripts contenidos en este repositorio, es necesario contar con Python 3.x instalado en el sistema. Los archivos pueden correrse directamente desde la terminal mediante:

python nombre_del_archivo.py
